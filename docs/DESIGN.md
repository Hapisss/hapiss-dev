# DESIGN.md

# Design Philosophy

The design should communicate confidence through simplicity.

Do not design to impress.
Design to communicate.

---

# Overall Feeling

The interface should feel:

* Premium
* Modern
* Calm
* Intentional
* Professional
* Minimal

The website should feel handcrafted rather than generated from a template.

---

# Design Principles

Every visual element must have a purpose.

Whitespace is preferable to visual clutter.

Typography is more important than decoration.

Hierarchy is more important than color.

Consistency is more important than creativity.

---

# Visual Direction

Inspired by:

* Apple
* Vercel
* Linear
* Raycast
* Stripe Documentation

These are references for quality and attention to detail, not templates to copy.

---

# Layout

Prefer generous spacing.

Use consistent alignment.

Maintain clear visual hierarchy.

Avoid crowded layouts.

Use grid systems consistently.

---

# Typography

Typography is the primary design element.

Prioritize:

* Readability
* Clear hierarchy
* Comfortable spacing
* Strong contrast

Avoid decorative typography.

---

# Color Philosophy

Use a restrained color palette.

Accent colors should guide attention, not dominate the interface.

Dark mode is the primary experience.

Light mode may be introduced later.

---

# Motion

Animations should support usability.

Good animations:

* explain transitions
* improve orientation
* provide feedback

Avoid animations that exist only because they look cool.

---

# Components

Every component should feel like it belongs to the same design system.

Buttons, cards, spacing, radius, shadows, and typography should all follow consistent rules.

Avoid one-off component designs.

---

# Accessibility

Accessibility is part of the design.

Maintain:

* sufficient contrast
* visible focus states
* keyboard accessibility
* semantic hierarchy

---

# Things to Avoid

* Glassmorphism
* Excessive gradients
* Particle backgrounds
* Heavy parallax
* Overly animated pages
* Generic portfolio templates
* Decorative UI with poor usability

---

# Definition of Good Design

The best compliment for this website is:

"It feels effortless."

Not:

"It has cool animations."
